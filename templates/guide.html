Anleitung

1. Daten eintragen

Tragen Sie zuerst die Daten für Absender, Empfänger sowie die restlichen Briefangaben ein. Felder, die in den meisten Briefen benötigt werden, sind farblich hervorgehoben.

2. Speichern wählen

Entscheiden Sie, ob die Angaben aus Absender und/oder Empfänger für spätere Aufrufe im Browser gespeichert werden sollen. Setzen Sie dafür die jeweiligen Speichern-Häkchen.

3. Haupttext schreiben

Wählen Sie im Bereich Textformat, ob Sie Markdown oder LaTeX verwenden möchten. Schreiben Sie danach den Brieftext im großen Textfeld.

4. PDF erzeugen

Klicken Sie auf PDF erzeugen. Falls ein Fehler auftritt, erscheint oben auf der Seite eine Meldung.
